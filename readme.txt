== Description ==

This plugin extends the WooCommerce e-commerce plugin by allowing to create custom price labels for products (like 'Call for Price' etc.).

This plugin was tested with WooCommerce 2.0.7.

== Installation ==

1. Upload the entire 'woocommerce-custom-price-label' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All options will be automatically added to product's edit page.

== Frequently Asked Questions ==

= Can I add custom price label before or after the price? =

You need [WooCommerce Custom Price Label Pro plugin](http://www.algoritmika.com/shop/wordpress-woocommerce-custom-price-label-pro-plugin/) to change these settings.

= Can I add custom price label between regular and sale prices? =

You need [WooCommerce Custom Price Label Pro plugin](http://www.algoritmika.com/shop/wordpress-woocommerce-custom-price-label-pro-plugin/) to change these settings.
