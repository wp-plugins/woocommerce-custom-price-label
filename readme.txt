Create any custom price label for any WooCommerce product.

== Description ==

WooCommerce Custom Price Label Plugin lets you add any price label to any WooCommerce product. Labels can be set **globally** for all products, or locally on **per product** basis.

== Installation ==

1. Upload the entire 'woocommerce-custom-price-label' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Custom Price Label.

== Frequently Asked Questions ==

= Can I add custom price label between regular and sale prices? =

You need [WooCommerce Custom Price Label Pro plugin](http://coder.fm/items/woocommerce-custom-price-label-plugin/) to change these settings.
